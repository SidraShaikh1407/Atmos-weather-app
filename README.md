# ATMOS — Weather App

A dark-themed weather app built with HTML, CSS, and JavaScript using the [WeatherAPI](https://www.weatherapi.com).

## Live Demo
🌐 [sidrashaikh1407.github.io/Atmos-weather-app](https://sidrashaikh1407.github.io/Atmos-weather-app/)

API key is pre-configured. Just open the link and search any city.

## Features
- Search weather by city
- Temperature in °C and °F
- Humidity, Wind, Visibility, UV Index, Pressure
- Air Quality Index (AQI)

## Tech
HTML · CSS · Vanilla JS · WeatherAPI
