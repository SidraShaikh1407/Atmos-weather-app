# ATMOS - Weather App

A dark-themed weather app built with HTML, CSS, and JavaScript using the [WeatherAPI](https://www.weatherapi.com).

## Features
- Search weather by city
- Temperature in °C and °F
- Humidity, Wind, Visibility, UV Index, Pressure
- Air Quality Index (AQI)

## Setup

1. Get a free API key from [weatherapi.com](https://www.weatherapi.com/signup.aspx)
2. In `index.html`, replace:
```js
const API_KEY = 'YOUR_WEATHERAPI_KEY_HERE';
```
3. Open with **Live Server** (VS Code) or any local HTTP server

> ⚠️ Don't open via `file://` — API calls will be blocked by the browser.
